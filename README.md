# ProjectFLOW

Plateforme de gestion et suivi des projets étudiants.

## Installation et lancement

```bash
git clone <url-du-repo>
cd ProjectFLOW
npm install
npm run dev
```

## Base de données

La base a changé : il faut supprimer l'ancienne et la recréer avec le nouveau schéma.
